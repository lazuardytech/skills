# Hive Setup

Setup skill for Hive Mind. Detects harness and creates agent config files.

## Usage

```
/hive-setup
```

## What it does

1. Detects current harness (Mastra Code, Command Code, OpenCode, Mistral Vibe)
2. Creates agent prompt files in the appropriate location
3. Reports status

## Install

```bash
npx skills add https://github.com/lazuardytech/skills --skill hive-setup
```
